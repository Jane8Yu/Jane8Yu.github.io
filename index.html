六月碎碎念

六月底估计要打工，估计到时候赶不回来写日记了，提前写了吧。
隔一两天没看书，阅读速度就立马坠崖式下降。
看《奇特的医生》前部分，非常敬佩柳比契夫，结合bujo、pomotodo，找到适合自己的办法。
别被潜意识欺骗，没兴趣是因为做得差，而不是没兴趣所以做得差，把它做好就好了。

2017-05-30

五月碎碎念

没有什么比感受不到危机更令人恐惧的了。
发现自己看的都是一些入门书，想要看专业级别的书，路漫漫其修远兮。
看了《硅谷》、《摔跤吧，爸爸》对大地怀有的敬畏之心、《加勒比海盗5：死无对证》我也想拥有一个指向心中所想的罗盘。
人体是最好的app。
将要去深圳打工，人身地不熟，对未知的焦虑。
沦为化学激素的奴隶，如何控制自己的不被化学激素奴役。
计算机技术与科学，给我一个机会去探索极致，把重叠错乱的重复交给计算机，解放自己的大脑用来思考，以有涯逐无涯的新技术方式。

Summary of 2017.3

This month I’ve been busy with internship-hunting.
I admit that I stll have a lot to learn, feeling so blue after the interview.

But sometimes I think is it ridiculours for foreign front-end engineers to be asked to use float css attribute to layout the page when css3 new attribute grid is available?
Using JavaScript prototype to implement oop is so ugly. Because JavaScript isn’t born for oop. Would it be more elegant to use delegate to reuse code?
Anyway, I just feel that some front-end tech and ideology is behind the times in here.
What is the best way to select the right one?
Can’t wait to see it!
